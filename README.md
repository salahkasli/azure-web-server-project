## 📌 Project Overview
This project demonstrates how I deployed a secure web server in the cloud using Microsoft Azure.

---

## 🛠️ Technologies Used
- Microsoft Azure
- Ubuntu Linux
- NGINX
- Virtual Network (VNet)
- Network Security Group (NSG)

---

## 🔧 What I Did
- Created a Virtual Machine in Azure
- Configured networking (VNet & Subnet)
- Secured the server using NSG rules
- Installed and configured NGINX
- Deployed a live web application

---

## 🔐 Security Implementation
- Restricted SSH access to my IP
- Opened only required ports (80, 22)
- Applied basic server hardening
